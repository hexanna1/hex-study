# Hex study tools

[Hex Study](https://hexanna1.github.io/hex-study/)
- [Patterns](https://hexanna1.github.io/hex-study/patterns.html)
- [Joseki](https://hexanna1.github.io/hex-study/joseki.html)
- [Openings](https://hexanna1.github.io/hex-study/openings.html)

This repo contains specs, tooling, website assets, and generated outputs for three Hex study workflows:

- Patterns: local pattern study, catalog generation, and pattern pages
- Joseki: local opening-line study and explorer
- Openings: whole-board opening study and explorer

This project expects a sibling [`../hexata`](https://github.com/hexanna1/hexata) checkout.

Repo layout:

- `plan/` specs and planning docs
- `docs/` website assets
- `results/` curated outputs
